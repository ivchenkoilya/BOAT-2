# Main Hero Bot — Reality 47

## Главное изменение

В Telegram-меню команд теперь регистрируется полный набор публичных функций бота, а не только шесть старых команд.

### Команды в группах

- `/start` — описание и запуск
- `/me` — своя роль и баланс
- `/who` — профиль участника по reply или `@username`
- `/stats` — подробная статистика
- `/top` — рейтинг беседы
- `/influence` — увеличить влияние
- `/hero_day` — Главный герой дня
- `/boss` — Центр Вселенной
- `/abilities` — способности ролей
- `/roles` — роли и звания
- `/fate` — Шар судьбы
- `/today` — типаж дня
- `/secret` — тайное задание
- `/ego 30` — Битва эго со ставкой
- `/sabotage` — саботаж
- `/impeachment` — импичмент
- `/rebellion` — бунт
- `/roulette` — рулетка влияния
- `/action` — проверяемое действие
- `/aura` — аура
- `/chsv` — уровень ЧСВ
- `/coin 20` — монетка на очки
- `/dice 20` — кубик на очки
- `/about` — описание бота
- `/feedback` — предложение разработчику
- `/help` — полный список команд
- `/status` — диагностика аналитики
- `/myid` — Telegram ID

Все новые команды имеют реальные обработчики и запускают те же механики, что и inline-меню.

## Области команд

- В группах и супергруппах Telegram показывает все 28 публичных команд.
- В личных сообщениях оставлены только `/start`, `/about`, `/feedback`, `/help` и `/myid`.
- Одновременно установлен запасной общий список, чтобы команды появились даже в клиентах, которые используют default scope.

## Установка на Amvera

Можно заменить только `main.py` и дождаться полной пересборки. Изображения босса и Шара судьбы менять не требуется.

После перезапуска открой беседу, введи `/` и при необходимости перезапусти Telegram: список команд иногда обновляется в клиенте не мгновенно.

Базу данных удалять не нужно.
