# Mini App «Центр Вселенной»

Фронтенд боевого рейда. Работает вместе с `main.py` Reality 49 и открывается через Direct Link Mini App.

- 4 боевые фазы по HP: 100–76%, 75–51%, 50–26%, 25–1%.
- Пятая картинка используется только после победы и свержения.
- До 4 участников одновременно.
- Личное здоровье, лечение, защита, ответные атаки босса.
- Ролевой урон, критические удары и уникальные способности.
- Итоговый экран: пьедестал топ-3 с наградами и полный список участников.
- Анимации, синтезированная музыка/звуки, вибрация Telegram.
