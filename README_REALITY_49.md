# Reality 49 · Mini App «Центр Вселенной»

## Главное

Команда `/boss` запускает общего босса, а карточка в группе получает кнопку **«⚔️ Войти в Mini App»**. Внутри приложения отображаются анимированный бой, четыре фазы по HP, здоровье игроков, лечение, защита, ролевые способности, журнал событий и итоговый рейтинг.

## Фазы

1. 100–76% HP — спокойная уверенность.
2. 75–51% HP — раздражение.
3. 50–26% HP — ярость.
4. 25–1% HP — разрушение эго.
5. 0 HP — отдельная плачущая фотография, показываемая только после победы.

## Участники и здоровье

- В бою одновременно участвуют максимум 4 человека.
- У каждого собственное количество HP, зависящее от роли.
- Босс отвечает раз в 30 секунд: атакует одного игрока, всю команду, ставит щит или временно запрещает атаки.
- При 0 HP участник временно выбивается, но может вернуться через кнопку лечения.
- Лечение восстанавливается 90 секунд, защита — 3 минуты.

## Роли

Отдельные значения обычного урона, критического шанса, критического множителя, максимального здоровья и способности настроены для:

- Декорации;
- Пыли;
- Массовки;
- Второстепенной роли;
- Временного Главного героя;
- Саботажного Главного героя;
- Честного Главного героя.

Общий кулдаун активной способности роли — 10 минут.

## Финал и награды

После победы Mini App и карточка в Telegram показывают пятую фотографию. Выводятся:

- 🥇 первое место — +250 очков;
- 🥈 второе место — +150 очков;
- 🥉 третье место — +100 очков;
- ниже — полный список всех участников и нанесённый ими урон без дополнительной награды.

## Настройка BotFather

1. Разместить проект на Amvera и получить HTTPS-адрес.
2. В `@BotFather` создать Mini App для бота через `/newapp`.
3. Указать URL: `https://АДРЕС-ПРОЕКТА/boss-app/`.
4. Задать short name `center` или другое значение.
5. То же значение записать в переменную `WEBAPP_SHORT_NAME`.
6. В Amvera добавить `BOT_TOKEN`, `BOT_USERNAME`, `WEBAPP_SHORT_NAME` и `WEBAPP_PUBLIC_URL`.

## Локальный просмотр интерфейса

```bash
python run_preview.py
```

Затем открыть:

```text
http://127.0.0.1:8765/boss-app/?demo=1
```

Демонстрационный режим работает без Telegram и показывает тестовый совместный бой. Боевые награды в нём не записываются в базу.

## База данных

При первом запуске Reality 49 сама добавляет новые поля в существующую SQLite-базу. Удалять базу и терять очки участников не нужно.
